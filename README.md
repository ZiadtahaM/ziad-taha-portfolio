# ziad-taha-portfolio

a frontend application solving state management and ui rendering complexities. it handles asynchronous data fetching and displays it in a responsive layout. the code focuses on component reusability and strict typing.

Tech stack: html, css, javascript, typescript, frontend frameworks

How to run: clone the repository, install dependencies if any, and open index.html in a browser or run the local development server.

Live demo: https://ZiadtahaM.github.io/ziad-taha-portfolio
